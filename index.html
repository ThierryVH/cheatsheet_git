CHEAT SHEET COMMANDES GIT

  • git initCrée un dossier de dépot dans le répertoire en cours.
  • git addAjouter un changement.
  • git commit -m "nom du commit"Valider ce changement et lui donner un nom.
  • git push origin masterEnvoyer les changements dans la branche "master".
  • git remote add origin [server] Connecter le dépôt local au dépôt sur serveur.
  • git checkout -b blablaCréer une branche nommée "blabla" et passer dessus pour l'utiliser.
  • git checkout masterRetourner sur la branche master.
  • git branch -d blablaSupprimer la branche "blabla".
  • git push origin blablaEnvoyer la branche "blabla" et la rendre disponible pour les autres.
  • git pullRécupérer et fusionner les changements distants.
  • git merge blablaPour fusionner la branche "blabla" avec celle active.
  • git diff blabla masterCompare le contenu des branches "blabla" et "master".
  • git checkout -- Fichier1Annuler les changements et revenir à la dernière sauvegarde.
